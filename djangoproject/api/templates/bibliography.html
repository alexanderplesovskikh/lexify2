Оформление библиографии

Общие правила оформления

Список литературы — это как визитная карточка вашей работы. Я видел, как научные руководители сразу ставят "неуд" за неправильный алфавитный порядок.

Самое смешное, что студенты тратят часы на поиск источников, но в последний момент не могут правильно расставить запятые в библиографии. Давайте сделаем так, чтобы у вас таких проблем не было.

Главное правило

Если вы не уверены в оформлении — лучше проверьте дважды. Неправильная библиография может перечеркнуть всю вашу работу.

Примеры оформления различных источников
Книга одного автора

Схема: Фамилия И.О. Название книги : подзаголовок / И.О. Фамилия. Место издания : Издательство, год. количество страниц.

Иванов И.И. Основы экономической теории : учебник / И.И. Иванов. М. : Вестник, 2023. 350 с.
Книга двух авторов
Петров П.П. Современный менеджмент : учебное пособие / П.П. Петров, С.С. Сидоров. СПб. : Вестник, 2023. 280 с.
Книга трех и более авторов

Схема: Название книги / И.О. Фамилия [и др.]. Место издания : Издательство, год. количество страниц.

Основы маркетинга / А.А. Алексеев [и др.]. М. : Вестник, 2023. 420 с.
Статья из журнала

Схема: Фамилия И.О. Название статьи // Название журнала. Год. № номер. С. страницы.

Статья из сборника

Козлов К.К. Проблемы современной экономики / К.К. Козлов // Актуальные вопросы экономики : сборник науч. тр. М. : Экономика, 2023. С. 45-52.

Важные правила и рекомендации

Алфавитный порядок

Источники располагаются строго в алфавитном порядке по фамилии первого автора или по названию

Нумерация

Каждый источник имеет порядковый номер, который используется для ссылок в тексте работы

Иностранные источники

Размещаются после русскоязычных источников в алфавитном порядке латинского алфавита

Дата обращения

Для интернет-ресурсов обязательно указывается дата последнего обращения к источнику

Шрифт и интервал

Times New Roman, 14 пт, полуторный интервал, выравнивание по ширине

Абзацный отступ

Первая строка каждого источника начинается с абзацного отступа 1,25 см

Часто задаваемые вопросы

Как оформлять источники без автора?

Если у источника нет автора, библиографическое описание начинается с названия. Например: "Экономика России : статистический сборник. М. : Росстат, 2023. 250 с."

Нужно ли указывать ISBN для книг?

ISBN указывается факультативно, но его наличие повышает точность библиографического описания. Указывается в конце описания: "ISBN 978-5-00000-000-0"

Как оформлять переводные издания?

Для переводных изданий указывается: "Смит Дж. Основы экономики : учебник / Дж. Смит ; пер. с англ. И.И. Иванова. М. : Экономика, 2023. 300 с."

Можно ли использовать Википедию в списке литературы?

Википедия не рекомендуется для научных работ как источник, поскольку информация может быть изменена любым пользователем. Лучше использовать первоисточники.
